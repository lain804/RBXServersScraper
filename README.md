# RBXServers Scraper

A simple Roblox vip server scraper for retrieving active private servers from a given place ID using rbxservers.xyz.

## 🚀 Quick Start

### Option 1 — pip
```bash
pip install -r requirements.txt
python main.py
```

### Option 2 - poetry
```bash
poetry install
poetry run python main.py
```
